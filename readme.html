This is the readme for the model associated with

Stinchcombe AR, Hu C, Walch OJ, Faught SD, Wong KY and Forger DB (2021) M1-Type, but Not M4-Type, Melanopsin Ganglion Cells Are Physiologically Tuned to the Central Circadian Clock. Front. Neurosci. 15:652996. doi: 10.3389/fnins.2021.652996

To run, enter ipRGCephys at the MATLAB command prompt. Simulated current-clamp voltage traces for M1 and M4 ipRCGs will be plotted as in Figure 1D. The value of Iapp is specified at the top of the file. This code was developed in MATLAB R2020b.
